# notes-app

Node application created for learning purposes

## Usage

Install:

```bash
	
npm install

```

Add note:

```bash
	
node app.js add --title="Note Title" --body="Note Body"

```

Read Note:

```bash
	
node app.js read --title="Note Title"

```

Remove Note:

```bash
	
node app.js remove --title="Note Title"

```
